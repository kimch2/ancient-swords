System requirements

Abyss Web Server is a multiplatform software designed for Microsoft Windows 95, 98, Millennium Edition (ME), NT 3.51 (and higher), 2000, XP (all editions), 2003, Vista (all editions), 7 (all editions), 2008 (all editions), Mac OS X 10.2 (and higher), and Linux (Kernel 2.4 and higher) systems. It is compatible with both 32 and 64-bit architectures and operating systems.

This guide documents the Windows version. Abyss Web Server for Windows minimum requirements are:

  • An Intel 80486 processor at 33 MHz.

  • 4 Mb of free RAM.

  • A modern web browser with HTML 4.0 and CSS 1.0 support.

  • TCP/IP protocol stack installed.

Although Abyss Web Server can run on a standalone computer, it is recommended to have a Modem or a network card and to be connected to a LAN (Local Area Network) or to the Internet.
